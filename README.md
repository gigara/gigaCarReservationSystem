# Car Rental Store Management System
Simple car rental store management system using Play Framework(Java 8) &amp; Angular 6 and mysql database

* Add vehicles
* Delete vehicles
* Reserve vehicles
* Return vehicles
* Availability check

# How to run
<code> 
  cd ui
</code>
<code>
  npm install
</code>
<code>
  cd ..
</code>
<code>
  sbt run
</code>

This will automatically start both play server & angular server.
